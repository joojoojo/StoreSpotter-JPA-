# 프로젝트 소개



> **StoreSpotter**은
> 전국 업종(편의점, 카페)/지역별 상권 분석을 통해 점포가 들어서기 좋은 상권을 추천해 주는 서비스입니다.

> SpringBoot, Spring Data JPA와 MySQL을 사용하여 개발하였고
> Docker, Github Actions, EC2, RDS, ALB를 통해 서버 및 자동 배포 파이프라인을 구축하였습니다.
> 

### **개발 기간 및 인원**

> 2024.01.~2023.04. (4개월)
백엔드 2명
